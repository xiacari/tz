Парсит указанный файл записи ответов сервера на отдельные ответы.
Для каждого ответа определяет его тип, размер, формат сжатия и тело ответа.
Если в теле присутсвует timestamp текущего (в момент ответа) времени, находит его и определяет время ответа.
Так же выводит статистику по типу запроса: кол-во и средний размер.

Как использовать:
1) Собрать и запустить приложение
2) Ввести команду "parse [путь до файла]"
Программа распарсит файл на отдельные ответы. После этого можно использовать команды "list", "info" и "response [индекс блока] <индекс ответа>".
3) Команда "list":
Выводит полный список блоков, из которых состоит исходный файл. В каждом блоке один или несколько ответов сервера. Для каждого блока выводит список его ответов. Для каждого блока и ответа выводит так же сжатую информацию о них.
4) Команда "info":
Выводит список типов поученных ответов, выстроенных по кол-ву. Для каждого типа выводит средний размер ответа.
Так же выводит время сессии, если было полученно хотя бы 2 ответа с timestamp полями в теле.
5) Команда "response [индекс блока] <индекс ответа>":
Выводит полную информцию об ответе, включая его тело.
6) При повторном использовании команды "parse", распарсится файл по новому пути, и последующие вызовы команд "list", "info" и "response" будут относиться к новому файлу.
7) Так же присутствуют утилитарные команды "help", "clear" и "exit".

Пример использования:
> tz.exe
> parse dumps/dump1
> list
> info
> response 0
> response 40 1
> exit
